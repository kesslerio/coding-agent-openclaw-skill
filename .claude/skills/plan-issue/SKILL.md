# Plan-Issue Skill

1. Gather context with read-only operations only.
2. Produce a concrete implementation plan with scope, sequence, assumptions, risks, and tests.
3. End with: `Reply APPROVE to execute this plan.`

Do not write files, install packages, commit, or open PRs in this skill.
